# node-hapijs
Exemplo de API em Node.js usando Hapi.js, curso realizado [Digital Inovation One](https://digitalinnovation.one).

## Instalar dependências
```
    yarn
```

## Neste exemplo utilizei o MONGO ATLAS

```
   yarn start
```
